# 异构 GPU 集群调度模拟器

异构 GPU 调度问题的主要挑战包括：
1) 计算能力差异：不同 GPU 的计算速度不同，任务在不同 GPU 上的执行时间差异较大，调度策略需考虑计算性能差异
2) 显存约束：任务对显存的需求不同，显存容量限制了任务的并发执行能力
3) 截止时间和优先级：不同任务可能有不同的截止时间与优先级，调度需平衡任务的紧急性与整体吞吐量

heterogeneous_gpu_scheduling_analysis.md 中有该问题的形式化建模与分析

算法设计：
• 基线方法（如 FIFO / SPT / EDF）。
• 改进的启发式方法（需给出伪代码与复杂度分析的文档）

实验要求：

• 指标：加权完成时间、平均完成时间、 deadline miss 数量、GPU 利用率。
• 比较算法： FIFO、 SPT、 EDF 和改进的启发式方法等。
• 小规模时可用 ILP/CP-SAT 求最优解进行对比。
• 使用 Python 进行仿真。
